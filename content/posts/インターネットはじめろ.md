## 新しいファイルを作る
コマンドプロンプトとかPowerShellで

```
cd C:\Users\omaenonamae\hugo\hogehoge
```
作業するフォルダーを移動を移動できるらしい
```
hugo new posts/hogehoge.md
```
で新しいファイルを作る  
すると、
```
content/posts/
```
に`hogehoge.md`って名前のファイルができる。  
ファイルを開くと、`draft: true`　みたいな項目があるので、公開するときはこれを`false`にする

